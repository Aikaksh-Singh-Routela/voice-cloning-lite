A lightweight Text-to-Speech web app supporting **60+ languages**. Convert any text to speech and download as MP3 — perfect for quick audio generation.

## 🚀 Features

| Feature | Description |
|---------|-------------|
| 🔊 **Text-to-Speech** | Convert any text to natural-sounding speech |
| 🌍 **60+ Languages** | Supports English, Spanish, French, German, Hindi, Chinese, and more |
| 🎚️ **Speed Control** | Normal or slow playback speed |
| 📥 **Download MP3** | Save generated audio to your device |
| ⚡ **Fast & Free** | Powered by Google's Text-to-Speech API |
| 📱 **Responsive** | Works on desktop and mobile |

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **TTS Engine:** gTTS (Google Text-to-Speech)
- **Deployment:** Streamlit Cloud
- **Language:** Python 3.11+

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Aikaksh-Singh-Routela/voice-cloning-lite.git
cd voice-cloning-lite

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
🎮 How to Use
Enter text in the text area

Select language from the dropdown

Choose speed (Normal or Slow)

Click "Generate Speech"

Listen to the audio

Download MP3 with one click
